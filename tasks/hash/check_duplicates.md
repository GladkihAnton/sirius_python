Дан целочисленный массив nums, верните true, если любое из значений встречается в массиве как минимум дважды, иначе верните false, если все значения в массиве различны.

 

Пример 1:
```
Ввод: nums = [1,2,3,1]
Вывод: true
```
Пример 2:
```
Ввод: nums = [1,2,3,4]
Вывод: false
```
Пример 3:

```
Ввод: nums = [1,1,1,3,3,4,3,2,4,2]
Вывод: true
```
