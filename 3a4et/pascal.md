дано число rowIndex, верните rowIndexth (0-indexed) строку паскаль треугольника

В Паскаль треугольнике, каждое числов это сумма двух предыдущих над ним.

Example 1:
```
Input: rowIndex = 3
Output: [1,3,3,1]
```
Example 2:
```
Input: rowIndex = 0
Output: [1]
```
Example 3:
```
Input: rowIndex = 1
Output: [1,1]
```
